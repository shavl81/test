# RICE + Value/Complexity Priority Tool

Десктопное приложение для приоритизации проектов с использованием методик RICE и Value/Complexity Matrix.

## Требования

- Python 3.8+
- PyQt6
- matplotlib
- SQLite3 (встроен в Python)

## Установка зависимостей

```bash
pip install PyQt6 matplotlib
```

## Запуск приложения

```bash
python rice_priority_tool.py
```

## Функционал

### 1. Ввод данных по методике RICE

Программа запрашивает следующие параметры:

- **Охват (Reach)** - количество пользователей/клиентов, которых затронет проект
- **Влияние (Impact)** - степень влияния на каждого пользователя (0.25-3.0)
- **Уверенность (Confidence)** - уверенность в оценках (0.0-1.0)
- **Усилия (Effort)** - требуемые ресурсы (человеко-месяцы или другие единицы)

RICE-балл рассчитывается по формуле: `(Reach × Impact × Confidence) / Effort`

### 2. Оценка ценности и сложности

- **Ценность (Value)** - ценность проекта для бизнеса (1-10)
- **Сложность (Complexity)** - сложность реализации (1-10)

Автоматическое определение квадранта:
- **Быстрая победа** - высокая ценность, низкая сложность
- **Стратегическая инициатива** - высокая ценность, высокая сложность
- **Заполнитель** - низкая ценность, низкая сложность
- **Избегать** - низкая ценность, высокая сложность

### 3. Дашборд

Приложение генерирует дашборд с:

- **График RICE-баллов** - столбчатая диаграмма для сравнения проектов
- **Матрица Value/Complexity** - scatter plot с проектами по квадрантам
- **Таблица проектов** - подробная информация по всем проектам

### 4. Работа с данными

- Сохранение проектов в SQLite базу данных
- Редактирование существующих проектов
- Удаление проектов
- Автоматическое обновление дашборда

## Структура БД

База данных `priority_projects.db` содержит таблицу `projects`:

| Поле | Тип | Описание |
|------|-----|----------|
| id | INTEGER | Первичный ключ |
| name | TEXT | Название проекта |
| reach | REAL | Охват |
| impact | REAL | Влияние |
| confidence | REAL | Уверенность |
| effort | REAL | Усилия |
| rice_score | REAL | RICE-балл |
| value | REAL | Ценность |
| complexity | REAL | Сложность |
| quadrant | TEXT | Квадрант |
| created_at | TEXT | Дата создания |
| updated_at | TEXT | Дата обновления |

## Использование

1. Введите название проекта
2. Заполните параметры RICE с подсказками
3. Укажите ценность и сложность
4. Нажмите "Сохранить проект"
5. Просмотрите результаты на дашборде
6. При необходимости отредактируйте проект через меню "Действия"

## Примечания

- Приложение работает на Windows, Linux и macOS
- Все данные сохраняются локально в файле БД
- Для корректного отображения графиков требуется установленная библиотека matplotlib
